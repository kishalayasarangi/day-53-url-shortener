# Day 53 — URL Shortener (Flask)

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A full stack URL shortener built with Flask and SQLite.
Shortens long URLs and tracks click counts.

## How to run
```bash
pip3 install flask
python3 app.py
```
Then open http://localhost:5000

## Features
- Shorten any URL instantly
- Custom short codes
- Click tracking and analytics
- QR code generation
- Top URLs leaderboard
